# Images Directory

Place your images here:

## Required Images

1. **logo.png** - Your company logo (recommended size: 200x60px, transparent background)
2. **favicon.png** - Browser favicon (16x16px or 32x32px)

## Optional Images

- Product screenshots
- Team photos
- Office images
- Any other marketing materials

## Image Optimization Tips

1. Use WebP format for better compression
2. Optimize images before uploading (use tools like TinyPNG)
3. Keep file sizes small for faster loading
4. Use appropriate dimensions for web display

## Logo Guidelines

- Format: PNG with transparent background
- Size: Max width 200px, height proportional
- File size: Keep under 100KB
- Colors: Should work well with orange theme
