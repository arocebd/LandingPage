# How to Add Your Dashboard Screenshot

## Quick Steps:

1. **Take a screenshot of your dashboard** (the one you showed me)
   - Make sure it shows the Dashboard Overview with stats
   - Best resolution: 1920x1080 or 1366x768

2. **Save the screenshot as**: `dashboard.png`

3. **Place it in this folder**: 
   ```
   LandingPage/assets/images/dashboard.png
   ```

4. **Refresh the landing page** - Your dashboard will appear animated in the hero section!

2. Press `Windows + Shift + S` to open Snipping Tool
3. Capture the main dashboard area (exclude browser chrome)
4. Save as `dashboard.png` in the images folder
